# 更新日志

[English](CHANGELOG.en.md) | 中文

本项目遵循语义化版本号的基本习惯：修复问题更新 patch，兼容新增功能更新 minor，破坏性变化更新 major。

## 未发布

- 将新增和编辑配置从主列表内联表单改为独立 Sheet，主卡片保持紧凑稳定；多规则表单一次只展开一条规则，标题与操作区固定，并在放弃未保存修改前确认。
- 将编辑器中的自动连接和自动重连设置整理为独立的自动化卡片，与基础字段和转发规则形成清晰层级。
- 隧道名称改为去除首尾空白、大小写不敏感的唯一标识；新增、编辑和导入均拒绝重名，JSON“作为副本”自动追加数字后缀。
- 优化本地监听状态探测：每轮只启动一次 `lsof`，运行中的本地/SOCKS 隧道保持 2 秒刷新，全部停止时降为 30 秒，并保留启动、停止、保存和手工刷新后的即时检查。
- 收紧本地数据边界：持久化配置和单个 SSH Config 文件读取上限为 1 MiB；真正打开 URL 前再次限制为带 Host 的 HTTP/HTTPS 地址。
- 增加连接组：同一个 SSH Host 可通过单个 SSH 进程混合承载多条本地、远程和动态 SOCKS 转发；新建时与 SSH Config 引用明确分型，保存后不能直接互转，每条规则以“启用此规则”独立控制是否参与连接，连接组统一启动、停止、重连及自动连接，并在启动前检查组内和跨组监听冲突。
- 配置导出升级为 `schemaVersion = 2`，继续只读导入 v1 单端口文件；已发布的单端口 `tunnels.json` 自动迁移为单规则连接组，首次写回前创建可恢复原始备份，导入会清除规则风险确认和自动启动。
- 增加 JSON 配置导入导出：可导出全部或所选配置，导入前校验格式、字段、数量、监听冲突和暴露风险；相同 ID 支持跳过、替换或作为副本，所有导入项关闭自动启动，并通过导入前备份、原子写入和失败回滚保护现有配置。
- 增加默认关闭的 macOS 登录项和逐隧道“应用启动时连接”：登录项使用 `SMAppService.mainApp`，以系统实际状态驱动开关且只显示需要处理的提示；手工启动与登录启动使用同一规则，只连接明确启用的配置，危险监听在没有有效确认时跳过，单条失败不阻塞其他配置。
- 增加 SSH Config 只读发现与批量导入：遍历可访问的 `Include`、去除重复别名，通过 `/usr/bin/ssh -G` 预览 `LocalForward`、`RemoteForward`、`DynamicForward` 及监听范围；检测到 `Match exec` 时先确认，导入只保存 Host 引用且不建立连接，并对可能对外暴露的监听保持风险确认。
- 增加默认关闭的连接失败与恢复通知；仅在用户启用时请求权限，每个连续故障周期最多发送一次失败通知和一次恢复通知，用户主动操作不触发掉线通知。
- 增加连接详情和“复制诊断信息”，展示状态时间、退出码、重试信息、错误类别与脱敏摘要；复制内容不包含配置名称、Host/IP、用户名、端口目标、私钥路径、完整命令或原始 stderr。
- 增加逐隧道、默认关闭的自动重连；可恢复故障按 2、5、10、30、60 秒退避，稳定运行 5 分钟后重置。
- 增加正在连接、等待网络和等待重连状态；断网与系统睡眠期间暂停重试，网络恢复并稳定 2 秒后只恢复一次。
- 使用运行代次隔离过期进程回调，手工停止会取消重试且不会被旧回调重新启动；认证、Host Key、端口冲突和配置错误不自动重试。
- 增加 OpenSSH 远程端口转发 `-R` 模式，支持远端监听、本地目标、非回环远端监听风险确认和旧配置恢复备份。
- 修复本地端口状态刷新阻塞主线程，以及应用处于非激活状态时菜单栏面板无法稳定显示的问题。
- 增加隧道标签、收藏、搜索、组合筛选、结果计数和清除筛选入口。
- 增加手工、名称、运行状态和最近使用时间排序；手工顺序与最近使用时间写入本地配置，保存失败时保持原状态，并兼容旧版 JSON 数组顺序。
- 标签会去除首尾空白并按大小写不敏感方式判重，最多保存 10 个标签，每个标签最多 32 个字符。
- 增加长标签横向滚动和按隧道模式搜索真实字段，并补充配置兼容与排序持久化测试。

## 0.3.2

- 修复分发包无法从标准的 `Contents/Resources` 加载 SwiftPM 本地化资源、导致脱离构建机后启动即退出的问题，并在 Release 校验中检查运行时资源布局。

## 0.3.1

- 删除隧道配置前增加二次确认，明确提示配置无法恢复，并在运行中的隧道将先停止连接时给出说明。
- 增加默认 Pull Request 模板、Issue 入口、贡献者行为准则和私密漏洞报告说明。
- 增加 GitHub Actions 依赖更新、自动生成 Release notes 及标签触发的自动 Release 工作流。

## 0.3.0

- 增加可自定义的全局快捷键，默认使用 `⌃⌥⌘T`；触发时显示并置前主界面，再次触发时关闭主界面。
- 增加快捷键录制、启用或停用、恢复默认和中英文设置界面；保存前检测系统可识别的冲突，保存失败时保留原设置。
- 菜单栏入口改为带悬浮提示的纯图标，减少菜单栏占用空间。
- 隧道列表可独立滚动，“添加隧道”和“退出”操作常驻面板底部。

## 0.2.1

- 对非回环本地监听地址增加明确风险确认，包括手动转发、动态 SOCKS 和 SSH Config 中的 `LocalForward`。
- 修复 SSH Config 校验未消费 stderr 管道可能导致阻塞或超时的问题。
- 增加非回环绑定确认、SSH Config 解析和相关本地化测试。

## 0.2.0

初始公开版本。

- 支持 macOS 菜单栏管理 SSH 本地端口转发。
- 支持动态 SOCKS 隧道。
- 支持复用 `~/.ssh/config` 中已有 `LocalForward` 的 SSH Config 模式。
- 支持英文和简体中文界面，默认跟随 macOS 系统语言。
- 使用本地 JSON 保存隧道配置，不保存服务器密码或私钥。
- 提供本机安装脚本和小范围 zip 分发脚本。
- 补充架构、分发、隐私、安全、贡献、发布和排障文档。
