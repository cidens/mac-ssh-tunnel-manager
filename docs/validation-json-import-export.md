# JSON 配置导入导出验收记录

本文记录 GitHub Issue #6“支持 JSON 配置导入导出”在 2026-07-19 完成的最终验收。测试只使用本机脱敏配置，不记录真实 Host、地址、端口、URL 或标签内容。

## 验收范围

- 导出全部或所选配置，并生成带格式版本、导出时间和应用版本的 UTF-8 JSON。
- 导入文件大小、数量、格式版本、字段、端口、URL、重复 ID、本地监听冲突及暴露风险校验。
- 相同 UUID 的跳过、替换和作为副本三种策略。
- 导入项关闭自动启动、清除最近使用时间，不保存临时风险确认。
- 导入预览不启动 SSH、不打开 URL、不操作登录项或通知权限。
- 导入前备份、原子写入，以及保存失败时恢复文件和内存。
- 中英文界面、原生文件窗口生命周期和菜单栏主面板交互。

## 自动化测试

核心测试覆盖：

- 导出顶层元数据和持久化字段白名单。
- 文件 1 MiB、配置 1000 条上限。
- 高版本 `schemaVersion`、重复 UUID、非法字段和非法 URL 拒绝。
- 三种相同 UUID 策略及副本新 UUID。
- 导入项 `isAutoStartEnabled = false`，其他允许的自动化设置保持。
- 回环别名、通配监听重叠、已有无关冲突和暴露监听提示。
- `tunnels.json.pre-import.bak` 创建、`0600` 权限与恢复。

应用层测试覆盖：

- 导入成功后不解析或启动 SSH，配置保持停止状态。
- 注入保存失败后，内存数组和磁盘文件都恢复到导入前状态。
- 系统保存/打开窗口被视为菜单面板流程的一部分。
- 文件窗口期间暂停“点击外部自动收起”，结束后的尾随全局点击不会关闭主面板。
- 中英文资源键和值一致。

## 构建验证

- 使用 `scripts/build-app-bundle.sh` 构建正式配置 `.app`。
- `CFBundleShortVersionString` 与 `CFBundleVersion` 均为 `0.3.2`。
- 可执行文件为 `arm64`。
- 英文、简体中文 Core 与 App 资源均存在。
- `codesign --verify --deep --strict` 通过。
- 覆盖安装到 `/Applications/SSH Tunnel Manager.app` 后正常启动。

该包仅用于 Issue #6 本机验收，不是公开 `v0.3.2` Release 资产。

## 手工验收

### 界面与文件窗口

- 主面板底部使用单一“导入”菜单，包含 SSH Config 导入和 JSON 配置导入导出，未增加横向按钮拥挤。
- 配置导入导出窗口正常打开，现有配置默认全选。
- 右下角仅关闭窗口的按钮显示“关闭 / Close”，不再使用可能暗示提交的“完成 / Done”。
- 清空导出选择时显示必须至少选择一项的原因；打开系统文件窗口期间显示进度和当前等待操作。
- 导出和导入文件窗口均能立即出现；点击取消后，配置导入导出窗口和主面板保持打开且可继续操作。

### 导出

- 选择一条脱敏配置导出到 `ssh-tunnel-manager-configs.json`，界面显示导出成功。
- 文件包含 `schemaVersion = 1`、`exportedAt`、`appVersion = 0.3.2` 和一条配置。
- 配置包含自动重连、自动启动、标签、收藏、最近使用时间和手工顺序等持久化字段。
- 文件不包含 PID、进程、stderr、错误历史、凭据或风险确认字段。
- 验收文件 SHA-256 为 `daa2a4e994a050739fadc5ef4a44401348757e37151d9a4859c3819a318445c0`。

### 导入与事务

- 导入刚导出的文件，默认“跳过”预览显示新增 0、替换 0、副本 0、跳过 1。
- 确认导入后显示成功提示，现有配置数量和内容不变。
- `tunnels.json` 与 `tunnels.json.pre-import.bak` 权限均为 `0600`，大小均为 3654 字节，SHA-256 均为 `d8da7231aa6d56f6ad7fa71eca9f16208d642501b486844933354dbe3f03d155`。
- 对同一文件切换“替换”时预览显示替换 1；切换“作为副本”时预览显示副本 1。两项仅验证预览，未提交，不向真实配置增加重复项。

## 验收结论

Issue #6 的导出范围、格式版本、隐私字段边界、导入限制、三种冲突策略、安全默认值、无副作用预览、事务备份与回滚、双语界面和菜单栏文件窗口交互均通过自动化与本机手工验收，满足功能验收条件。
